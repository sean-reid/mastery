# Mastery

An interactive ranking of 100 human skills by difficulty to master, with cited sources and playable challenges.

**[View the site](https://sean-reid.github.io/mastery/)**

## What it is

A single-page visualization scoring 100 skills across 19 dimensions of difficulty. Every score has a cited source. Twelve interactive challenges let you attempt fragments of each skill yourself.

## Skills

From chess and cardiac surgery to Vinkensport and mokume-gane. A mix of famous anchors, niche crafts, and genuine obscurities across ten categories.

## Built with

Astro, GSAP, Lenis, Web Audio API, Canvas, and server-rendered SVG.
